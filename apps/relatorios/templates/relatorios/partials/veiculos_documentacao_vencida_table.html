Funcionalidade em Desenvolvimento

O relatório de veículos com documentação vencida estará disponível quando os campos de data de vencimento forem adicionados ao modelo Veículo.

Para adicionar este relatório:

  • 1. Adicionar campos de vencimento na tabela Veiculo: vencimento_ipva, vencimento_seguro, vencimento_licenciamento, etc.
  • 2. Implementar lógica de consulta para documentos vencidos ou próximos ao vencimento (ex: próximo mês)
  • 3. Calcular dias restantes para vencimento
